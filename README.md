# calculator
#### Simple Calculator in Python

## Overview

This is a simple command-line calculator developed in Python. The program allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. It runs continuously until the user chooses to exit.

## Features

* Addition of two numbers
* Subtraction of two numbers
* Multiplication of two numbers
* Division of two numbers
* Handles division by zero errors
* Menu-driven interface
* Continuous execution using a loop until the user exits

## Technologies Used

* Python 3

## How It Works

1. The user enters two numbers.
2. A menu of operations is displayed.
3. The user selects an operation:

   * 1 → Addition
   * 2 → Subtraction
   * 3 → Multiplication
   * 4 → Division
   * 5 → Exit
4. The program performs the selected operation and displays the result.
5. The process repeats until the user selects the Exit option.

## Error Handling

* If the user attempts to divide by zero, the program displays:
  Error: Division by zero is not allowed.
